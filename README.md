# 📚 Study Planner (Java)

A simple console-based Study Planner application built using Java.

## 🚀 Features
- Add study tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Save tasks permanently using file handling

## 🛠 Technologies Used
- Java
- OOP Concepts
- File Handling
- ArrayList

## ▶ How to Run

1. Navigate to src folder:
   cd src

2. Compile:
   javac StudyPlanner.java

3. Run:
   java StudyPlanner

## 📂 Data Storage
All tasks are stored in data.txt file.
